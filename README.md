# IWD 2026 Hackathon

![BioGraph Logo](BioGraphApp/brand/bioGraph_logo.png)

This repository contains **BioGraph**, a personal health and habit tracking Android app built at the International Women's Day 2026 Hackathon.

## Repository Structure

- [**BioGraphApp/**](BioGraphApp/) — the full Android project (source code, build files, and detailed [README](BioGraphApp/README.md))
- **specs/** — feature specifications used to plan and build the app
- **Syne/** — the Syne typeface (OFL-licensed) used in the app's branding

## Built with GitHub Copilot and SpecKit

This app was developed using **GitHub Copilot** with the **SpecKit** workflow. Each feature was planned through a structured spec-driven process — writing detailed specifications, implementation plans, and task breakdowns before any code was written.

The **specs/** folder contains the full set of feature specifications:

| Spec | Feature |
|---|---|
| [001-add-track-data](specs/001-add-track-data/) | Core data type creation and daily calendar tracking |
| [003-rich-data-types](specs/003-rich-data-types/) | Scale (0–5) and multiple choice input types |
| [004-calendar-insights](specs/004-calendar-insights/) | AI-powered correlation insights panel |
| [005-brand-theme-refresh](specs/005-brand-theme-refresh/) | Green brand theme, logo, and Syne typography |
| [006-insight-sorting](specs/006-insight-sorting/) | Sort insights by strength or by data type |
| [007-mock-data-trends](specs/007-mock-data-trends/) | Realistic mock data generation and trend indicators |

Each spec folder includes a feature specification, implementation plan, task list, data model, and requirements checklist.
